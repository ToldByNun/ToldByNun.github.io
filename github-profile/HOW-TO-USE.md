# So aktivierst du dein GitHub Profil-README

GitHub zeigt automatisch eine `README.md` auf deiner Profilseite an, wenn du ein Repository erstellst, dessen Name **exakt deinem Username** entspricht.

## Schritt-für-Schritt

1. Auf GitHub einloggen (`@ToldByNun`).
2. Neues Repo erstellen: <https://github.com/new>
   - **Repository name:** `ToldByNun` (genau so, gross/klein wie dein Username)
   - **Public** muss aktiv sein
   - Haken bei "Add a README file" setzen
3. Klicke auf das neue Repo, dann auf `README.md` -> Bearbeiten (Stift).
4. Den kompletten Inhalt aus `README.md` (in diesem Ordner) reinkopieren.
5. "Commit changes" klicken.

Fertig. Geh auf <https://github.com/ToldByNun> -> dein Profil zeigt jetzt das README.

## Tipp

Das README nutzt externe Bilder (Capsule-Render, Stats, Trophies, Activity-Graph). Wenn etwas nicht laedt, ist meist ein Adblocker oder ein downed Service schuld — dann den betreffenden Block entfernen.

Private Repos: Links zu privaten Repos wuerden fuer andere Nutzer **404** sein. Im README sind sie deshalb nur als **Beschreibung** gelistet.

## Tipp (Stats)

Falls dir Stats / Trophies / Activity zu viel sind, einfach die entsprechenden Abschnitte in `README.md` loeschen.

Wenn du Farben / Akzent aendern willst:
- `6EE7B7` ist der Akzent-Gruen (gleicher wie auf der Portfolio-Seite)
- `0d1117` ist die GitHub-Dark-Hintergrundfarbe
